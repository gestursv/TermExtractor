This package includes 41 term collections from the Icelandic Term Bank in 41 separate files in TBX format.
The following files are also included:
- ABOUT.TXT: General information about the Icelandic Term Bank
- LICENCE_CC-BY-SA-30.txt: The Creative Commons licence that covers all 41 term collections.
- TBX_STATS.TXT: Information about number of terms in each file and number of words in different languages.
